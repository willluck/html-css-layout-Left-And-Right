# html-css-layout-Left-And-Right
**CSS基础实践指导系列**

## CSS经典左右布局（左边固定，右边自适应或右边固定，左边自适应方法总结归纳）
- 方法一：左侧div浮动 右侧元素设置margin-left自适应
- 方法二：左侧div浮动，右侧元素创建BFC
- 方法三：左侧div采用绝对布局，右侧元素设置margin-left自适应
- 方法四：外层父级元素使用table布局，子元素使用table-cell
- 方法五：双float + calc计算宽度
- 方法六：flex布局
